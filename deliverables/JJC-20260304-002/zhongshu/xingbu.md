# 刑部交付（JJC-20260304-002）

## 1. 目标
建立可量化验收体系，验证 NATS 协同桥可靠性与性能指标。

## 2. 测试矩阵（纲要）
- 功能测试
  - 双向消息路由正确性
  - 主题映射一致性
  - 适配器回执闭环
- 可靠性测试
  - 重试机制有效性
  - 幂等去重正确性
  - DLQ 落盘完整性
- 性能测试
  - 吞吐（RPS）
  - 延迟（P50/P95/P99）
- 安全测试
  - 未授权主题访问拦截
  - 非法签名拒绝

## 3. 验收标准
- 端到端成功率 >= 99%
- P95 延迟 < 200ms（内网基线）
- 异常消息可进入 DLQ 且可追溯
- trace_id 全链路贯穿

## 4. 核心用例（样例）
1) OpenCode 下发命令 -> OpenClaw 执行 -> 回执成功
2) OpenClaw 回传失败 -> Bridge 重试3次 -> DLQ
3) 重复 message_id 注入 -> 仅首条生效

- 《测试计划》
- 《压测报告》
- 《验收签核单》
