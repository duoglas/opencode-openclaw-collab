## 2) `/oc-reconfigure` 查看/修改当前配置

命令（查看）：
```text
/oc-reconfigure --show
```

截图式输出（示例）：
```text
path: /home/duoglas/.config/ocbridge/env
current config (sanitized):
{
  "NATS_URL": "nats://100.95.183.80:4222",
  "OC_NODE_ID": "homepc-1",
  "NODE_ID": "homepc-1",
  "CAPABILITY": "coding",
  ...
}
usage: /oc-reconfigure KEY=value [KEY=value ...]
```

命令（修改）：
```text
/oc-reconfigure OC_NODE_ID=homepc-2 NODE_ID=homepc-2 RUN_TIMEOUT=1200
```

截图式输出（示例）：
```text
updated: OC_NODE_ID, NODE_ID, RUN_TIMEOUT
path: /home/duoglas/.config/ocbridge/env
updated config (sanitized):
{
  "OC_NODE_ID": "homepc-2",
  "NODE_ID": "homepc-2",
  "RUN_TIMEOUT": "1200",
  ...
}
```

## 3) `/oc-events --follow` 与 `/oc-follow` 实时刷新

命令（短跟随30秒）：
```text
/oc-events --follow --seconds=30 --limit=50
```

或：
```text
/oc-follow --seconds=30
```

截图式输出（示例）：
